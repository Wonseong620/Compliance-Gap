[Project Overview]

This repository contains the dataset and replication materials for the study:

  | Aid Intensity and the Commitment–Compliance Gap in Human Rights

The project examines whether higher levels of Net Official Development Assistance (ODA) per capita are associated with a divergence between:

- Formal human rights commitments (ICCPR ratification)
- Observed political repression outcomes (Political Terror Scale)

The central construct is the Commitment–Compliance Gap (Gap A), which captures repression conditional on treaty commitment.
